# Tableau HR Analytics Dashboard

## Project Overview

This project presents a comprehensive Human Resources Analytics Dashboard built using Tableau. The dashboard transforms raw HR employee data into actionable insights that help organizations understand workforce composition, compensation patterns, employee demographics, and workforce growth trends.

The goal of this project is to demonstrate data visualization, dashboard design, and business intelligence skills by answering critical HR-related questions through interactive visual analytics.

---

## Business Problem

HR departments often manage large volumes of employee data but struggle to extract meaningful insights for decision-making.

This dashboard helps stakeholders:

* Understand workforce distribution
* Analyze employee demographics
* Evaluate salary structures
* Identify top-performing earners
* Monitor employee growth trends
* Track leave balances
* Explore workforce characteristics interactively

---

## Key Questions Answered

### 1. How many people are in each job role?

Provides employee count by job title to understand workforce allocation.

### 2. What is the gender distribution?

Analyzes workforce diversity based on gender.

### 3. What is the age spread of employees?

Shows age distribution to identify workforce demographics.

### 4. Which jobs pay more?

Compares average salaries across different job roles.

### 5. Who are the top earners in each job category?

Identifies highest-paid employees by role.

### 6. Qualification vs Salary Analysis

Examines the relationship between employee education level and salary.

### 7. Staff Growth Trend (Last 5 Years)

Tracks hiring trends and workforce expansion over time.

### 8. Employee Filter by Starting Letter

Interactive employee search based on the first letter of employee names.

### 9. Leave Balance Analysis

Evaluates leave availability and usage across employees.

### 10. Quick HR Dashboard

Provides an executive-level summary of key HR metrics.
